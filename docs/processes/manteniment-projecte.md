# Manteniment del Projecte

## Objectiu

Definir les tasques periòdiques necessàries per mantenir el repositori ordenat i actualitzat.

---

# Freqüència

Revisió recomanada:

- després de cada bloc important
- abans d'una release
- mensualment

---

# Revisar documentació

Verificar:

- enllaços
- coherència
- duplicats
- informació obsoleta

---

# Revisar estructura

Comprovar:

- carpetes
- fitxers
- nomenclatura

---

# Revisar prompts

Eliminar:

- redundàncies
- prompts obsolets
- inconsistències

---

# Revisar plantilles

Actualitzar quan sigui necessari.

---

# Revisar recursos

Comprovar:

- snippets
- checklists
- exemples

---

# Revisar GitHub

Verificar:

- README
- CHANGELOG
- ROADMAP
- TODO

---

# Millora contínua

Registrar:

- lliçons apreses
- noves bones pràctiques
- coneixement reutilitzable

---

# Resultat esperat

Repositori net, coherent i fàcil de mantenir.
