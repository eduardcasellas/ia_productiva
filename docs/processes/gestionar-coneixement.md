# Gestionar Coneixement

## Objectiu

Transformar l'experiència adquirida en coneixement reutilitzable.

---

# Fonts

El coneixement pot provenir de:

- projectes
- errors
- automatitzacions
- revisions
- experiència acumulada

---

# Classificació

Separar entre:

## Temporal

- SESSION
- TODO

## Permanent

- Knowledge Base
- Glossary
- Documentation

---

# Validació

Abans d'afegir nou coneixement comprovar:

- que sigui reutilitzable
- que no estigui duplicat
- que estigui verificat

---

# Actualització

Quan sigui necessari:

- ampliar documentació
- crear snippets
- crear checklists
- crear plantilles

---

# Resultat esperat

El framework aprèn de manera contínua.
