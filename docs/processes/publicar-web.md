# Publicar Web

## Objectiu

Definir el procés estàndard per publicar una pàgina web o una actualització.

---

# Abans de publicar

Verificar:

- funcionalitat
- rendiment
- enllaços
- SEO
- responsivitat

---

# Revisió tècnica

Comprovar:

- errors de consola
- recursos inexistents
- temps de càrrega
- accessibilitat

---

# Revisió de contingut

Verificar:

- ortografia
- coherència
- estructura
- crides a l'acció

---

# Publicació

Seguir el procediment del proveïdor d'allotjament.

Verificar que el desplegament finalitzi correctament.

---

# Validació

Comprovar:

- pàgina principal
- formularis
- navegació
- dispositius mòbils

---

# Tasques posteriors

Actualitzar:

- CHANGELOG
- ROADMAP (si cal)

---

# Resultat esperat

Web publicada sense errors i completament funcional.
