# Crear Agent

## Objectiu

Definir un procés estàndard per crear un nou agent d'IA reutilitzable dins d'IA-Productiva.

Un agent és un rol especialitzat amb una responsabilitat concreta i un comportament ben definit.

---

# Entrades

Abans de començar cal definir:

- objectiu de l'agent
- tipus de problemes que resoldrà
- públic objectiu
- eines disponibles
- limitacions

---

# Procés

## 1. Definir el rol

Descriure:

- què fa
- què no fa
- quan s'ha d'utilitzar

---

## 2. Definir les responsabilitats

Llistar les funcions principals.

Evitar responsabilitats duplicades amb altres agents.

---

## 3. Definir les entrades

Quina informació necessita.

Per exemple:

- documentació
- especificacions
- dades
- context

---

## 4. Definir les sortides

Què ha de generar.

Per exemple:

- informe
- codi
- document
- estratègia
- revisió

---

## 5. Definir el comportament

Especificar:

- to
- nivell tècnic
- metodologia
- criteris de qualitat

---

## 6. Validar

Comprovar:

- reutilització
- claredat
- absència de duplicats

---

# Resultat esperat

Un agent especialitzat, reutilitzable i coherent amb la resta del framework.
