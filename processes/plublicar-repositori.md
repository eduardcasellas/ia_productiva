# Publicar Repositori

## Objectiu

Definir el procés oficial per publicar una nova versió del repositori a GitHub.

---

# Preparació

Abans de publicar verificar:

- documentació actualitzada
- estructura coherent
- sense fitxers temporals
- TODO revisat
- CHANGELOG actualitzat

---

# Revisar canvis

Executar:

```bash
git status
```

Revisar els fitxers modificats.

---

# Afegir els canvis

```bash
git add .
```

---

# Crear el commit

Utilitzar un missatge descriptiu.

Exemple:

```bash
git commit -m "Complete documentation processes"
```

---

# Publicar

```bash
git push
```

---

# Verificació

Comprovar a GitHub:

- commits visibles
- fitxers actualitzats
- documentació correcta

---

# Resultat esperat

Repositori sincronitzat i disponible amb tots els canvis publicats.
