# Git

## Clonar

```bash
git clone <repositori>
```

## Estat

```bash
git status
```

## Afegir

```bash
git add .
```

## Commit

```bash
git commit -m "Missatge"
```

## Publicar

```bash
git push
```

## Actualitzar

```bash
git pull
```

## Crear branca

```bash
git checkout -b nova-branca
```

## Canviar de branca

```bash
git checkout main
```
